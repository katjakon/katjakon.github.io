# Using Mycroft: Installation & Creating Skills

{: .warning } 
> To install Mycroft, you need to have Linux installed.

## Downloading and Installing Mycroft
Please check out the instructions on their website. At least for me on Linux, this worked fine! There is also the possibility to install Mycroft on a Raspberry Pi. We have not tested it on a virtual machine for Windows. <br>

[Get Mycroft](https://mycroft-ai.gitbook.io/docs/using-mycroft-ai/get-mycroft){: .btn .btn-primary .fs-5 .mb-4 .mb-md-0 .mr-2 }

## Creating your first skill
[Documentation](https://mycroft-ai.gitbook.io/docs/skill-development/introduction/your-first-skill)
To create your first skill, run the following command from command line.<br>
```mycroft-msk create```<br>
This will guide you through all the necessary steps interactively.
